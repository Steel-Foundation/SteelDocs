Steel uses the decompiled source code from Minecraft to have a reference. A guide on how to decompile Minecraft can be found [here](../../development/decompile-minecraft).

During contributing to this project some standards are needed to be followed. This will grantee that the code will be unified and everyone has easier access to the code base. These code standards can be found [here](../../development/code-standard).
