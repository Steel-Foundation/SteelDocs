## Is there an Egg?
Currently there is no egg for provider, to host steel somewhere. Also steel is currently in pre alpha, so shouldn't be used on a server.

## Where is the jar?
Steel is not written in Java, so it doesn't have a jar. You can only find an exe, linux, or mac executable. Which you can find in our homepage or github repo.

## Will be steel vanilla parity?
Short and simple that is the main goal

## Has steel plugins?
Currently not, steel is in WIP and plugins are planned but currently not possible.

## I have a problem what should I do?
You can report at [github](https://github.com/Steel-Foundation/SteelMC/issues) or join our Discord(https://discord.gg/MwChEHnAbh) and report it there in the bugs channel.
