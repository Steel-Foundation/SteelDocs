Here you will find answers to most of the common questions!

## How do I install Steel?
You can find the full guide [here](../running).

## Is there an Egg?
Currently there is no egg for a provider to host Steel somewhere. Also, Steel is currently in pre-alpha, so it shouldn't be used on a server.

## Where is the jar?
Steel is not written in Java, so it doesn't have a jar. You can only find a Windows, Linux, or Mac executable, which you can find on our homepage or GitHub repo.

## Will Steel have vanilla parity?
Short and simple: that is the main goal.

## Does Steel have plugins?
Currently not. Steel is a WIP, and plugins are planned but currently not possible.

## I have a problem, what should I do?
You can report it on [GitHub](https://github.com/Steel-Foundation/SteelMC/issues) or join our [Discord](https://discord.gg/MwChEHnAbh) and report it in the bugs channel.
