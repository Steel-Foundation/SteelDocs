## Конфигурация сервера в SteelMC

Настройте свой сервер SteelMC с помощью пользовательских параметров и функций.

## Доступные руководства

- [**Ссылки сервера**](../server-links) - настройка ссылок сервера, отображаемых в меню многопользовательской игры.

## Справочная документация

Для полных технических справок смотрите:

- [**Конфигурация сервера**](../server-configuration) - все параметры сервера для `config/config.toml`
- [**Конфигурация мира**](../world-configuration) - все параметры мира для `config/worlds.toml`
- [**Флаги функций**](../../../reference/feature-flags) - флаги времени компиляции для настройки вашей сборки
